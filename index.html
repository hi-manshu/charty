    üöÄ Quick Start

    Get started with Charty in just a few minutes:

    @Composable
fun MyChart() {
    BarChart(
        dataCollection = listOf(
            BarData(10f, "Jan"),
            BarData(20f, "Feb"),
            BarData(15f, "Mar")
        ),
        config = BarChartConfig(
            showAxis = true,
            showGridLines = true
        )
    )
}

    üìä Chart Types

    Charty supports a wide variety of chart types:

    Bar Charts

    • Bar Chart - Standard vertical bars
    • Horizontal Bar Chart - Horizontal orientation
    • Stacked Bar Chart - Multiple series stacked
    • Comparison Bar Chart - Side-by-side comparison
    • Lollipop Bar Chart - Lollipop style visualization
    • Waterfall Chart - Cumulative effect display
    • Wavy Chart - Bars with wavy tops
    • Bubble Bar Chart - Bars with bubble indicators
    • Mosaic Bar Chart - Mosaic pattern bars
    • Span Chart - Range-based visualization

    Line Charts

    • Line Chart - Connect data points with lines
    • Area Chart - Filled area under the line
    • Multiline Chart - Multiple data series
    • Stacked Area Chart - Stacked filled areas

    Point Charts

    • Point Chart - Scatter plot visualization
    • Bubble Chart - Size-based data points

    Other Charts

    • Pie Chart - Circular sector representation
    • Radar Chart - Multi-axis visualization
    • Multiple Radar Chart - Compare multiple datasets
    • Candlestick Chart - Financial data visualization
    • Combo Chart - Combine multiple chart types
    • Block Bar - Block-style bars
    • Circular Progress - Circular progress indicator

    üé® Customization

    Every chart in Charty is highly customizable:

    • Colors: Define your own color schemes
    • Styles: Customize line thickness, bar width, corner radius, etc.
    • Animations: Control animation duration and style
    • Axes: Configure axis labels, grid lines, and formatting
    • Labels: Full control over data labels and legends
