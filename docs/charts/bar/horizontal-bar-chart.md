# Horizontal Bar Chart
A bar chart with horizontal orientation for better label readability.
## Basic Usage
```kotlin
HorizontalBarChart(
    dataCollection = listOf(
        BarData(100f, "Category A"),
        BarData(200f, "Category B"),
        BarData(150f, "Category C")
    )
)
```
## When to Use
- Long category labels
- Ranking or comparison data
- Mobile displays with limited width
