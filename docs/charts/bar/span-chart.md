# Span Chart

Visualize ranges or time spans.

## Basic Usage

```kotlin
import com.himanshoe.charty.bar.SpanChart
import com.himanshoe.charty.bar.data.SpanData

SpanChart(
    dataCollection = listOf(
        SpanData(
            start = 10f,
            end = 50f,
            label = "Task 1"
        )
    )
)
```
