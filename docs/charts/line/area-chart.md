# Area Chart

Line chart with filled area below the line.

## Basic Usage

```kotlin
import com.himanshoe.charty.line.AreaChart
import com.himanshoe.charty.line.data.LineData

AreaChart(
    dataCollection = listOf(
        LineData(10f, "Jan"),
        LineData(20f, "Feb"),
        LineData(15f, "Mar")
    )
)
```
